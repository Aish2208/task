# TaskBoard 🚀

A sleek, drag-and-drop task management dashboard built with React. Features include:

## ✨ Features
- Drag & drop tasks between columns (To Do → In Progress → Done)
- Subtask progress tracking with animated progress rings
- Priority-based color coding
- Tag system with dynamic colors
- Search & filter by priority
- Inline editing & task creation
- Due date tracking with overdue warnings
- Responsive design
- Dark cyberpunk aesthetic

## 🛠️ Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)
